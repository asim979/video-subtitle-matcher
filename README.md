A PowerShell script that automatically renames .srt subtitle files to match video file names (e.g., for proper recognition by VLC or other media players).
<br><br>

📁 Step 1: Prepare the Folder
Make sure both video files (e.g., .mp4, .mkv) and their corresponding subtitle files (.srt) are in the same folder.

<br><br>

⚡ Step 2: Open PowerShell
Go to the folder that contains your video and subtitle files.

Right-click inside the folder and select “Open in Terminal” or “Open PowerShell window here.”
